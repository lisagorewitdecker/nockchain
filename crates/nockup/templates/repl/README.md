# {{project_name}}

Interactive REPL-style NockApp generated by `nockup`.

{{description}}

## Files

- `src/main.rs`: Rust runtime and line-oriented REPL loop.
- `hoon/app/app.hoon`: app entry kernel compiled to `out.jam`.
- `hoon/common/wrapper.hoon`: wrapper core.
- `hoon/lib/lib.hoon`: local Hoon library.
- `build.rs`: optional `hoonc` compile hook used during Cargo builds.

## Build and run

From the workspace directory that contains `nockapp.toml`:

```sh
nockup project build {{project_name}}
nockup project run {{project_name}}
```

From this project directory directly:

```sh
cargo build --release
hoonc hoon/app/app.hoon
cargo run --release
```

## Notes

- The runtime expects `out.jam` in the project root at startup.
- The REPL prompt is `repl>`.
